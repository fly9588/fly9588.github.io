讓 CB 執行較順暢的方法(可能)
將下列文字複製至文字檔後，修改副檔名為 *.bat 作為批次檔，
每當啟動 CB 前就執行這個再執行 CB 即可刪除些暫存檔。
以下範例 CB 安裝路徑為 C，請自行依照自己的系統與環境去修正相關路徑。
============================== 複製以下文字 ==============================
@echo off
title 刪除快取並跳過 patch 執行 CB
@echo --------------------------------------------------
@echo 按任意按鍵後開始刪除
@echo 包含以下於 programs 內的暫存檔案：
@echo 副檔名為 *.twit
@echo clanicon 資料夾與其內容檔案
@echo img 資料夾與其內容檔案
@echo polygon 資料夾與其內容檔案
@echo queue 資料夾與其內容檔案
@echo skin 資料夾與其內容檔案
@echo --------------------------------------------------
pause
@echo 刪除 *.twit 檔案
del /q "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\*.twit"
@echo 刪除 clanicon 資料夾與其內容檔案
rmdir /q /s "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\clanicon"
@echo 刪除 img 資料夾與其內容檔案
rmdir /q /s "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\img"
@echo 刪除 polygon 資料夾與其內容檔案
rmdir /q /s "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\polygon"
@echo 刪除 queue 資料夾與其內容檔案
rmdir /q /s "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\queue"
@echo 刪除 skin 資料夾與其內容檔案
rmdir /q /s "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\skin"
@echo 按任意按鍵後執行 CB
pause
cd "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\"
start "" "C:\Program Files (x86)\CyberStep\CosmicBreak_TW\programs\cosmic.exe" -launch
exit
============================== 複製以上文字 ==============================

※原理與副作用：
此功能是大量刪除 CB 內的各種暫存檔案，就如同字面上面所述，
副作用為每當進機庫、副機庫時，skin 都得重新讀取過一次，請自行斟酌使用。
畢竟你已經把暫存檔砍了，他必然會重新讀取，但總比你堆著放些用不到的暫存檔好。
不經由 patch 視窗直接執行 CB
使用此方法前，當下必須更新到最新的版本才能進行遊戲，
將 CB 目錄下的資料夾 programs 中的 cosmic.exe 建立一個捷徑至桌面後，
將其捷徑內容的目標與起始文字修改成下列敘述：

目標：
"D:\Program Files\CyberStep\CosmicBreak_TW\programs\cosmic.exe" -launch
起始位置：
"D:\Program Files\CyberStep\CosmicBreak_TW\"

之後就可以直接執行捷徑略過 patch 直接執行 CB 了。
